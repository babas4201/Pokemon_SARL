Le fichier zip est composé de quatre parties:

1) BD

bd2.sql se trouve la base de donnée vierge
insert_personne & insert_pokemon ainsi que Remplissage_pokemon_lien remplissent les bases personne et pokemon

Pour construire la base de données entière avec des données déja insérées:
Executer le script "pokemon_placement.sql"

2) Dossier

Se trouve dans ce répertoire le dossier à rendre en extension .docs ou en PDF.

3) Modélisation BD

Il y a dans ce repertoire les diagrammes de classes, et le modèle physique de données fait avec Power Designer

4) Website

Ici se trouve le site web en lui-même prévu pour être executé en localhost.
La page d'accueil est le fichier "index.php".
Les différents fichiers sont logiquement nommés pour retrouver facilement les fichiers correspondants.
Les images sont stockées dans le dossier "img".
Les devis et factures en PDF sont respectivement stockés dans les dossiers "devis" et "facture " avant l'envoie par mail.
La librairie HTML2PDF est placée dans un dossier spécial.
